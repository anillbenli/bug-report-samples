# Bug Report

Title: Payment page crashes on invalid credit card

Environment:
- Browser: Chrome
- OS: Windows 11
- Device: Desktop

Steps to Reproduce:
1. Login to account
2. Add product to cart
3. Go to payment page
4. Enter invalid credit card details
5. Click Pay

Expected Result:
Error message should appear and user should not be charged

Actual Result:
Page crashes and user is redirected to homepage

Severity: High
